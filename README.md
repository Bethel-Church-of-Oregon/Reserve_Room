# 벧엘교회 장소 예약 시스템

교회 회의실 및 장소를 온라인으로 예약하고 관리할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **캘린더 현황** — 주간/월간 보기, 확정 예약(솔리드)과 승인 대기(빗금) 구분 표시
- **장소별 색상** — 15개 장소 각각 다른 색상으로 구분
- **예약 신청** — 누구나 비밀번호 없이 신청 가능 (15분 단위 시간 선택)
- **중복 방지** — 동일 장소·시간 충돌 시 즉시 안내
- **관리자 모드** — 비밀번호 입력 후 승인/거절/삭제 처리
- **반응형 디자인** — 데스크탑, 태블릿, 스마트폰 모두 지원

## 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) |
| 언어 | TypeScript |
| 스타일 | Tailwind CSS |
| 데이터베이스 | SQLite (`better-sqlite3`) |

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 환경 설정

`.env.local` 파일에서 관리자 비밀번호를 변경합니다.

```env
ADMIN_PASSWORD=원하는비밀번호
```

기본값은 `bethel2024`입니다.

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:8000](http://localhost:8000) 으로 접속합니다.

### 4. 프로덕션 배포

```bash
npm run build
npm start
```

## 데이터베이스

- SQLite 파일은 `data/reservations.db`에 저장됩니다.
- 앱 최초 실행 시 자동으로 생성되며, 15개 장소 기본 데이터가 입력됩니다.
- `data/*.db`는 `.gitignore`에 등록되어 있어 git에 커밋되지 않습니다.
  → git pull/push 시 서버의 예약 데이터가 유지됩니다.

## 화면 구성

### 메인 캘린더 (`/`)

- 현재 예약 현황을 주간/월간 캘린더로 확인
- 확정된 예약: 진한 색상 블록
- 승인 대기 중인 예약: 빗금 패턴
- 우측 상단: **장소 예약 신청** / **관리자 모드** 버튼

### 장소 예약 신청 (`/reserve`)

| 항목 | 필수 |
|------|------|
| 제목 | ✅ |
| 장소 | ✅ |
| 날짜 | ✅ |
| 예약 시간 (15분 단위) | ✅ |
| 담당자 | ✅ |
| 기타 노트 | — |

신청 후 관리자 승인 시 확정 처리됩니다.

### 관리자 모드 (`/admin`)

비밀번호 입력 후 사용 가능합니다.

- **승인 대기** 탭: 체크박스로 복수 선택 후 일괄 승인, 또는 거절(사유 입력 필수)
- **승인 완료** 탭: 확정된 예약 목록 확인 및 삭제
- **전체** 탭: 모든 예약 내역 조회
