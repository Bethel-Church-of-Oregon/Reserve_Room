# 벧엘교회 장소 예약 시스템

교회 회의실 및 장소를 온라인으로 예약하고 관리할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **캘린더 현황** — 주간/월간 보기, 확정 예약(솔리드)과 승인 대기(빗금) 구분 표시
- **장소별 색상** — 20개 장소 각각 다른 색상으로 구분
- **장소 필터** — 장소 필터 패널에서 원하는 장소만 선택해 캘린더에 표시 (멀티 선택)
- **예약 신청** — 누구나 비밀번호 없이 신청 가능 (15분 단위 시간 선택)
- **반복 예약** — 매일/매주/매월 반복 예약 지원, 충돌 날짜 자동 제외
- **중복 방지** — 동일 장소·시간 충돌 시 즉시 안내
- **이메일 알림** — 관리자 승인/거절 시 신청자 이메일로 자동 안내 발송 (일괄 승인 시 요약 이메일)
- **관리자 모드** — 비밀번호 입력 후 승인/거절/삭제 처리
- **반응형 디자인** — 데스크탑, 태블릿, 스마트폰 모두 지원

## 기술 스택

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 환경 설정

`.env.local` 파일을 설정합니다.

```env
# Postgres 연결 (Vercel Marketplace에서 Neon 연동 시 자동 주입)
POSTGRES_URL=postgres://...

# 관리자 비밀번호 (기본값: bethel2024)
ADMIN_PASSWORD=원하는비밀번호

# Gmail 앱 비밀번호 (이메일 알림 기능에 필요)
GMAIL_APP_PASSWORD=발급받은앱비밀번호
```

> **로컬 개발:** Neon 대시보드 또는 Vercel 프로젝트 설정에서 `POSTGRES_URL`을 복사해 `.env.local`에 붙여넣으세요.

#### Gmail 앱 비밀번호 발급 방법

1. [myaccount.google.com](https://myaccount.google.com) → **보안**
2. **2단계 인증** 활성화 (필수)
3. 검색창에 "앱 비밀번호" 검색 → **앱 비밀번호 생성**
4. 앱 이름 입력 후 생성 → 16자리 비밀번호를 `GMAIL_APP_PASSWORD`에 입력 (공백 없이)

> `GMAIL_APP_PASSWORD`를 설정하지 않으면 이메일 발송이 건너뛰어지며, 예약 승인/거절 자체는 정상 동작합니다.

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:8000](http://localhost:8000) 으로 접속합니다.

### 4. 프로덕션 배포

```bash
npm run build
npm start
```

## 데이터베이스

- **Vercel Postgres (Neon)** 사용 — Vercel 서버리스 배포에 적합.
- Vercel 프로젝트에서 [Neon](https://vercel.com/marketplace/neon) 연동 시 `POSTGRES_URL` 자동 주입.
- 앱 최초 실행 시 테이블 자동 생성, 20개 장소 시드 데이터 입력.

## 화면 구성

### 메인 캘린더 (`/`)

- 현재 예약 현황을 주간/월간 캘린더로 확인
- 확정된 예약: 진한 색상 블록 / 승인 대기: 빗금 패턴
- **장소 필터**: "장소 필터" 버튼을 눌러 패널을 펼치고, 원하는 장소 chip을 클릭해 필터링 (멀티 선택, 전체 보기로 초기화)
- 우측 상단: **장소 예약 신청** / **관리자 모드** 버튼

### 장소 예약 신청 (`/reserve`)

| 항목 | 필수 |
|------|------|
| 제목 | ✅ |
| 장소 | ✅ |
| 날짜 | ✅ |
| 예약 시간 (15분 단위) | ✅ |
| 담당자 | ✅ |
| 이메일 | ✅ |
| 반복 (없음/매일/매주/매월) | — |
| 반복 종료일 | — (반복 시 필수) |
| 기타 노트 | — |

신청 후 관리자 승인 시 확정 처리되며, 이메일로 결과가 안내됩니다.

### 관리자 모드 (`/admin`)

비밀번호 입력 후 사용 가능합니다.

- **승인 대기** 탭: 체크박스로 복수 선택 후 일괄 승인, 또는 거절(사유 입력 필수)
- **승인 완료** 탭: 확정된 예약 목록 확인 및 삭제
- **전체** 탭: 모든 예약 내역 조회
